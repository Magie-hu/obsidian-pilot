## Risk Control

1. **Code isolation:** Separate open-source core and paid modules into different repos
2. **No forced payment:** Basic features permanently free, only monetize heavy automation
3. **No vendor lock-in:** All local file operations, privacy-first positioning
4. **Continuous iteration:** Adjust product direction based on user feedback

# Quick Start Guide

## Installation

```bash
pip install obsidian-pilot
```

## Initialize LLM-Wiki Knowledge Base

```bash
obsidian-pilot init /path/to/vault -t llm-wiki
```

## Import Notes

```bash
obsidian-pilot import /path/to/vault --apply
```

## Automate Links

```bash
obsidian-pilot link /path/to/vault --apply
```

# Templater Template Usage Guide

## Installation

1. Open Obsidian Settings
2. Community Plugins → Browse
3. Search Templater → Install → Enable

## Configuration

1. Settings → Templater
2. Template location: `_templates`
3. Enable global hotkey

## Built-in Templates

Obsidian Pilot provides:

- `raw-material.md` — Raw materials
- `wiki-entry.md` — Wiki entries
- `index.md` — Index page

## Tips

- Ctrl+P to quickly create new notes
- Select template to auto-generate frontmatter
- Variable substitution supports dates, titles, etc.
